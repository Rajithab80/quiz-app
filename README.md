Interactive Quiz Application

## Description

This is a **learning-based project** developed to understand core concepts of **HTML, CSS, and JavaScript**.
The project was built by following tutorials and reference materials, and then practiced independently to strengthen frontend development skills.

## Purpose

* To practice JavaScript logic and DOM manipulation
* To understand event handling and dynamic UI updates
* To gain hands-on experience with frontend development

## Technologies Used

* HTML
* CSS
* JavaScript

## Features

* Multiple-choice questions
* Score calculation
* Interactive user interface
* Simple and responsive design

## How to Run the Project

1. Download or clone the repository
2. Open the project folder in **VS Code**
3. Open `index.html` using **Live Server** or directly in a browser

## Disclaimer

> This project was created for **learning and practice purposes only**.
> Code logic and structure were developed by referring to tutorials and online resources to improve understanding of frontend concepts.

## Learning Outcome

* Improved understanding of JavaScript fundamentals
* Hands-on experience with real-time UI interaction
* Better familiarity with frontend project structure
